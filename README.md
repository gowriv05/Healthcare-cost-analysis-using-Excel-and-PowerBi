# Healthcare-cost-analysis-using-Excel-and-PowerBi


##  Project Overview
This project analyzes healthcare cost data to understand the gap between hospital charges, Medicare payments, and patient burden. The goal is to identify cost inefficiencies and improve transparency in healthcare pricing.

---

##  Objectives
- Analyze hospital-wise and state-wise healthcare costs
- Identify hospitals with high cost gaps
- Compare covered charges vs actual payments
- Understand patient financial burden

---

##  Files Included
1. Raw Dataset
2. Cleaned & Transformed Dataset (Power Query)
3. Power BI Dashboard Report

---

##  Tools & Technologies
- Excel-Power Query (Data Cleaning & Transformation)
- Power BI Desktop
- DAX (Data Analysis Expressions)

---

##  Data Cleaning Steps
- Removed null and duplicate values
- Standardized column names
- Converted data types
- Created calculated columns (Cost Gap, Patient Burden)

---

##  Key Metrics (KPIs)
- Average Covered Charges: $36.13K
- Average Medicare Payment: $8.49K
- Average Total Payment: $9.71K
- Average Patient Burden: 15.35%
- Total Discharges: 7M
- Average Cost Gap: $26.43K

---

##  Dashboard Insights

###  Hospital Analysis
- Certain hospitals have extremely high cost gaps
- A small number of hospitals contribute disproportionately to overall costs

###  State Analysis
- States like AK, DC, and HI show higher Medicare payments
- Patient payments vary significantly across states

###  Cost Gap Analysis
- Large gap between covered charges and actual payments
- Indicates pricing inefficiencies in healthcare system

###  Patient Burden
- Patients pay around 15% of total cost on average
- Even small percentages translate into high financial burden

---

##  Key Findings
- Healthcare pricing lacks transparency
- Medicare payments are significantly lower than hospital charges
- Cost gaps are concentrated in a few hospitals
- Patients still face considerable financial burden

---

##  Conclusion
This analysis highlights the need for better cost transparency and pricing regulations in the healthcare system. Reducing cost gaps can help improve affordability and trust for patients.
